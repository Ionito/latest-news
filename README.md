# Latest news

Latest news is a simple site for showing and searching the news get from [News Api](https://newsapi.org/)

## Running Locally

```bash
pnpm i
pnpm dev
```

## Using

- Vite + React + Typescript
- TanStack Router

## Vercel

Seted Vercel to build on push to main
